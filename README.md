# Load Cell Measure

## Overview

This project provides tools for reading, logging, and visualizing load cell data through a serial connection and CSV plotting interface.

## Files

- **serial_read_load_cell.py** - Serial communication module for reading data from load cell sensors
- **csv_plotter_gui.py** - GUI application for plotting and analyzing CSV data

## Features

- Real-time load cell data acquisition
- Serial port communication
- CSV data export and import
- Interactive data visualization

## Requirements

- Python 3.x
- Required libraries listed in project dependencies

## Usage

Run the GUI application:

```bash
python csv_plotter_gui.py
```

Or use the serial reader:

```bash
python serial_read_load_cell.py
```
